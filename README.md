# AI-Powered Data Analysis Platform

## 🚀 Overview
This project allows users to upload datasets and analyze them using natural language queries.

## 💡 Features
- Upload CSV dataset
- Automatic table creation (ETL pipeline)
- Natural language → SQL using LLM
- Dynamic schema detection
- Data visualization (charts)
- AI-generated insights

## 🛠 Tech Stack
- Python
- MySQL
- Pandas
- Matplotlib
- Groq API (LLaMA 3.1)
